# HelpWords

Визуальный тренажёр по французским словам и выражениям для ориентации в пространстве.

## Live

- GitHub Pages: <https://polpal1623.github.io/HelpWords/>
- Репозиторий: <https://github.com/PolPal1623/HelpWords>

## Что внутри

- 15 выражений в словаре.
- 10 раундов и 6 вариантов ответа в каждом вопросе.
- Иллюстрации из общей картинки-спрайта `ChatGPT Image.png`.
- Переключаемый режим перевода для облегчённого повторения.
- Финальный обзор ошибок и результата.

## Запуск

1. Откройте `index.html` в браузере.
2. Приложение полностью статическое и не требует сборки.

## Редактирование

- Словарь хранится в массиве `vocabulary`.
- Позиции иллюстраций задаются в `buildScene`, а объяснения находятся в `buildExplanation`.
- Дизайн синхронизирован с остальными приложениями French Learning Suite.
